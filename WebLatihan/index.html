Pengalaman Belajar di Dicoding

Awal saya belajar dicoding adalah ketika melihat story instagram teman-teman kampus saya membagikan pengalaman belajarnya, mulai dari skil coding hingga mendapatkan sertifikat. Saya sangat jelaous kepada teman saya, ketika dia menceritakan pengalaman belajarnya, dan dia sangat jago dikampus ketika belajar pemograman mobile (Android). Sampai-sampai banyak teman cewek meminta bantuan dia untuk membantu mengerjakan tugas. Menurut dia, belajar Learning path belajat Android di Dicoding sangat bagus, Bahkan kemarin teman saya telah mendapatkan sertifikat Google Certified - Associate Android Developer karena telah mnyelesaikan kelas Android Expert. Hal ini membuat saya sangat tertarik dengan dicoding, saya pun mencoba peruntungan dengan mengikuti kelas-kelas Gratis terlebih dahulu karena tekendala biaya.

Ketika membuka instagram, saya melihat salah satu postingan Dicoding ada program Studi Independen di Dicoding, dan saya pun mencoba mendaftar, kapan lagi kan ya.. bisa belajar Gratis dengan mengikuti SIB. Saat melihat Learning path WEB Developer Dicoding, saya sangat tertarik dan mendaftar SIB dengan paket belajar Front End dan Back End. Jujur saja saya sangat tertarik dengan Back-End, tapi jika diberikan kesempatam menguasai Front-End juga Why not.
